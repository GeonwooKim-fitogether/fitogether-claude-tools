SECTION 01

제품 개요

iPad Sunshield는 야외 훈련 현장에서 EPTS(선수 추적 시스템) 데이터를 확인하는 코칭 스태프를 위해 하드웨어팀이 직접 개발한 차광(遮光) 후드 + 전용 삼각대 거치 시스템입니다. 아이패드를 햇빛과 열로부터 보호하고, 코치 눈높이에 맞춰 화면을 안정적으로 거치합니다.

PROBLEM — 현장의 문제

한여름 필드에서 아이패드가 꺼진다

실시간 EPTS 데이터를 확인하는 도중 태양열로 인한 아이패드 강제 종료(과열)가 발생하고, 강한 직사광선의 화면 반사로 데이터 판독 자체가 어려워 훈련 분석이 중단되는 일이 반복되었습니다.

SOLUTION — 우리의 해법

열 차단 + 시인성 + 이동 편의까지

열을 차단하고 시인성을 확보하는 전용 후드를 설계하고, 장비 이동이 잦은 코칭 스태프를 위해 이동·수납 편의성을 극대화한 전용 거치 시스템까지 하나의 패키지로 구축했습니다.

한눈에 보는 제품 폴딩 구조

1폴딩 상태 — 컴팩트 보관
2전개 — 별도 도구 불필요
3사용 상태 — 아이패드 장착
POINT  외부는 화이트(열 반사), 내부는 블랙(빛 반사 차단)의 듀얼톤 설계가 이 제품의 가장 큰 시각적 특징입니다. 고객에게 제품을 소개할 때 가장 먼저 보여줄 수 있는 차별점입니다.

핵심 특징 Key Features

FEATURE A

차열 & 시인성 최적화 — Dual-Tone Design

색상 하나하나가 기능입니다. 보기 좋은 투톤이 아니라, 열과 빛을 각각 해결하는 설계입니다.

외부 White
화이트 마감으로 직사광선의 열 흡수를 최소화 → 기기 과열(강제 종료) 방지
내부 Black
블랙 마감으로 빛 반사를 원천 차단 → 실내 수준의 선명한 데이터 화면 제공
FEATURE B

컴팩트 폴딩 & 인케이스 수납 — Integrated Storage

사용 후 즉시 접을 수 있는 폴딩 구조로 부피를 최소화했습니다. 접은 선쉴드는 핏투게더 하드케이스 내부 Dock 하단의 전용 수납공간에 완벽하게 들어가도록 설계되어, 별도의 적재 공간이 전혀 필요하지 않습니다.

수납 위치
하드케이스 내부 Dock 하단 전용 공간 — 기존 장비 패킹 동선 그대로 유지
FEATURE C

코치 친화적 이동 시스템 — Mobility Solution

내구성이 뛰어난 전용 삼각대 소프트 케이스가 기본 포함되며, 가방 후면에는 하드케이스(캐리어) 손잡이에 끼울 수 있는 캐리어 핸들 슬롯이 설계되어 있습니다.

기대 효과
코치가 한 손으로 하드케이스와 삼각대를 동시에 운반 — 필드 이동·장비 패킹 시 피로도 획기적 감소

조립 및 사용 방법

본 제품은 별도의 도구 없이 현장에서 신속하게 조립할 수 있는 툴리스(Tool-less) 및 마그네틱 결합 방식을 채택했습니다. 전체 과정은 4단계로, 처음 사용하는 코치도 몇 분 안에 설치를 마칠 수 있습니다.

1
아이패드 체결 및 선쉴드 결합
실리콘 홀더 → 마그네틱 부착 → 구조 확인

아이패드 삽입 — 전용 실리콘 홀더(Silicone Holder)를 가볍게 벌려 아이패드(11″/10.9″)를 정위치에 끼워 넣습니다.
마그네틱 부착 — 접혀 있는 선쉴드를 펼친 후, 내부 중앙의 가이드에 맞춰 실리콘 홀더를 가져다 댑니다. 강력한 내장 자석(Magnetic Mount)에 의해 ‘딸깍’ 소리와 함께 정위치에 고정됩니다.
구조 확인 — 실리콘 홀더가 선쉴드 안쪽에 안정적으로 안착되었는지 확인합니다.
TIP  이후 이동 시에는 이 상태 그대로 선쉴드를 접어 아이패드를 보호할 수 있습니다. 별도로 아이패드를 분리할 필요가 없습니다.

분해 — 훈련 종료 후 분해는 조립의 역순으로 진행합니다.

SECTION 05

비즈니스팀 핵심 포인트

고객(구단·코칭 스태프) 커뮤니케이션 시 아래 세 가지 가치를 중심으로 설명해 주세요.

VALUE 1 — 신뢰성
한여름에도 끊기지 않습니다
화이트 외장이 태양열 흡수를 막아 아이패드 과열 종료를 방지하고, 블랙 내장이 반사를 차단해 실내 수준의 시인성을 확보합니다.
VALUE 2 — 효율성
짐이 늘어나지 않습니다
폴딩 후 하드케이스 Dock 하단 전용 공간에 수납. 기존 장비 패킹 동선에 그대로 흡수되어 별도 적재 공간이 필요 없습니다.
VALUE 3 — 현장성
코치 혼자, 한 손으로 끝납니다
툴리스·마그네틱 조립으로 도구가 필요 없고, 삼각대 가방을 캐리어 손잡이에 끼워 한 손으로 모든 장비를 운반합니다.

자주 묻는 질문 (FAQ)

Q1어떤 아이패드와 호환되나요?
iPad Pro 11형, iPad Air 10.9형 전용입니다. 그 외 모델은 실리콘 홀더 규격이 맞지 않을 수 있으니 도입 전 모델 확인이 필요합니다.
Q2조립에 공구가 필요한가요?
아니요. 툴리스(Tool-less) 설계와 마그네틱 결합 방식으로 별도 도구 없이 현장에서 즉시 장착할 수 있습니다. 퀵 릴리스 플레이트 볼트도 손으로 조입니다.
Q3잔디밭처럼 고르지 않은 지면에서도 안정적인가요?
네. 야외 잔디밭과 불규칙한 지면을 전제로 설계된 광폭 지지대형 삼각대를 제공하며, 코칭 스태프 눈높이에 맞춰 최대 160cm 이상까지 높이를 조절할 수 있습니다.
Q4이동할 때 짐이 하나 더 늘어나는 것 아닌가요?
아닙니다. 선쉴드는 접어서 핏투게더 하드케이스 내부 전용 공간에 수납되고, 삼각대 가방은 하드케이스 손잡이에 끼워져 한 손 운반이 가능합니다. 추가 적재 공간이 필요 없습니다.
Q5훈련 중 자리를 옮길 때마다 다시 조립해야 하나요?
퀵 릴리스 플레이트 시스템으로 선쉴드-삼각대의 체결·분리가 신속하게 이루어지며, 아이패드는 선쉴
